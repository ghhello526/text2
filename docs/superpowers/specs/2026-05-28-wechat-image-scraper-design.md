# 微信公众号文章图片批量下载工具

## 目的
一个轻量级 Python 脚本，用于批量下载微信公众号文章中的图片。

## 整体架构
该工具由一个主脚本（`main.py`）构成。它从本地文本文件（`urls.txt`）中读取 URL 列表，逐一请求每个 URL，解析 HTML 提取文章标题和图片地址，并将图片下载到结构化的目录中（`images/<文章标题>/`）。

## 模块划分
1. **URL 加载模块**：读取 `urls.txt`，过滤空行，返回可遍历的 URL 列表。
2. **页面解析模块**：
   - 使用 `requests` 库并携带标准浏览器 `User-Agent` 获取文章 HTML。
   - 使用 `BeautifulSoup` 提取 `<title>`，并对其进行 Windows 文件路径合法化处理（将 `\/:*?"<>|` 替换为 `_`）。若标题缺失，则使用当前时间戳作为回退名称。
   - 定位所有包含 `data-src` 属性的 `<img>` 标签（微信公众号的懒加载机制），提取真实图片 URL。
3. **下载模块**：
   - 创建 `images/` 基础目录，并为每篇文章标题创建子目录。
   - 遍历提取到的图片 URL，请求图片二进制数据，按序号保存（如 `01.jpg`、`02.jpg`）。

## 错误处理
- **网络错误**：页面请求或图片下载过程中的超时或 HTTP 错误将被捕获，打印日志到控制台，并跳过该项以允许批量任务继续执行。
- **元素缺失**：若 `data-src` 缺失或页面结构发生变化，则优雅地跳过该图片或文章。
